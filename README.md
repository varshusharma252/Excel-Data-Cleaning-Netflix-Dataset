## Objective
To clean and standardize a real-world dataset using Excel by handling missing values, validating data types, and maintaining raw vs cleaned data separation.

## Dataset
Netflix Movies and TV Shows dataset

## Tools Used
- Microsoft Excel

## Cleaning Steps Performed
- Identified and highlighted missing values
- Validated uniqueness using show_id
- Standardized text fields using TRIM and PROPER
- Converted text-based dates into ISO format (YYYY-MM-DD)
- Maintained separate raw and cleaned datasets
- Documented issues in Data_Quality_Notes

## Files Included
- Raw_Data.xlsx
- cleaned_dataset.csv

## Outcome
A clean, analysis-ready dataset prepared following professional data cleaning practices.
